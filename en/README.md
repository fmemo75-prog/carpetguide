# 📊 Rug & Kilim Comparison Matrix (English)

This table serves as a quick-reference guide for different types of rugs.

| Rug Type | Origin | Material | Knot Type | Durability | Value |
| :--- | :--- | :--- | :--- | :--- | :--- |
| **Hereke** | 🇹🇷 Turkey | Silk | Ghiordes (Double) | High | Investment |
| **Uşak** | 🇹🇷 Turkey | Wool/Cotton | Ghiordes | Medium | Decorative |

### 🔍 Technical Details (Quick View)

| Feature | Handmade (Art) | Machine Made (Utility) |
| :--- | :--- | :--- |
| **Back Side** | Uneven knots, visible pattern | Perfectly uniform, plastic-like |
| **Fringes** | Part of the rug foundation | Sewn on later |
| **Dyes** | Natural (Root dyes) | Synthetic (Chemical) |
| **Longevity** | 50 - 100+ Years | 5 - 10 Years |
